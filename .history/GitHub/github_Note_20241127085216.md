## 正常的代码拉取合并方法
```// 查看远程
git remote -v 
// 获取远程指定分支到本地临时新建的分支
// 获取远程master的分支的代码到临时新建的temp
git fetch origin master:temp
// 查看版本差异
// 查看temp分支与当前分支的差异
git diff temp 
// 将临时分支temp合并到当前分支
git merge tmep
// 删除临时分支
git branch -D temp
```
