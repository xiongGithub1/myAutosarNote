# CARSim 模拟器学习

## 功能

- 求解运算 ，司机在车上，通过控制车速，方向，刹车等，模拟出车况。 车辆的动态特性。
- 可视化的后处理  视频 曲线图
- 提供很好的Gui界面

https://blog.csdn.net/YOURUOLI/article/details/127245122

同一个libraries下面有多个Database，一个Database中有多个Dataset.

Database 就相当于小学课程， Dataset 就相当于课程 英语 数学，Libraries就相当于数学老师，英语老师可以访问英语老师教过的所有课程，可以是数学老师访问教过的数学课程，也可以是物理老师访问教过的物理课程。

run Control也是一个最基本的Libraries。
